# 💬 NexChat - Real Time Chat App

NexChat is a modern real-time chat application built using the MERN Stack and Socket.IO, providing fast and seamless communication between users. It features secure authentication, live messaging, profile image upload with cropping, and a premium responsive user interface.
---

# 🚀 Features

✅ User Authentication (Login & Register)

✅ JWT Authentication

✅ Real-time Messaging using Socket.IO

✅ One-to-One Private Chat

✅ Online / Offline User Status

✅ Last Seen Status

✅ Separate Chats & Contacts Sections

✅ Premium Modern UI

✅ Responsive Design

✅ Profile Image Upload

✅ Image Cropping Support

✅ Sidebar User List

✅ Auto Scroll Chat Messages
